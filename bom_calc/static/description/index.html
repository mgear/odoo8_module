Create new BOM calc . Select Product and BOM.
Drag Start or End task.
Change "Calc Purch Qty" and Click "Calc"
Change "Calc only products" : where select in product: Can be Purchased.
Routes: Select Routes for product.
For Example if have product with route: BUY and want calc only that. Just select and calc.
