Email Auto-Resend

Resend failed emails automatically

This module give you the possibility to configure the number of attempts to resend a failed email automatically.

How to use

After installing this module you will find a new line in the System Parameters with the key 'mail_attempt_send'. Change the value to set the number of attempt to automatically resend a failed email.
